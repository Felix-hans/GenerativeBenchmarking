Here's the implementation of the `moveZeroes` function in Python:

```python
class Solution:
    def moveZeroes(self, nums):
        """
        Do not return anything, modify nums in-place instead.
        """
        # Initialize two pointers: one for the non-zero elements and one for the current element
        non_zero_ptr = 0
        
        # Iterate through the array
        for i in range(len(nums)):
            if nums[i] != 0:
                # Swap the current element with the non-zero element pointed by non_zero_ptr
                nums[i], nums[non_zero_ptr] = nums[non_zero_ptr], nums[i]
                non_zero_ptr += 1
```
