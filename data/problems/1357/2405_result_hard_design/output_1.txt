Sure! Here's the implementation of the `Cashier` class in Python:

```python
from typing import List

class Cashier:
    def __init__(self, n: int, discount: int, products: List[int], prices: List[int]):
        self.n = n
        self.discount = discount
        self.products = products
        self.prices = prices
        self.customer_count = 0

    def getBill(self, product: List[int], amount: List[int]) -> float:
        self.customer_count += 1
        bill = 0.0
        for i in range(len(product)):
            p = product[i]
            a = amount[i]
            idx = self.products.index(p)
            bill += self.prices[idx] * a

        if self.customer_count % self.n == 0:
            bill = bill - (bill * self.discount / 100)

        return bill
```

You can create an instance of the `Cashier` class and use the `getBill` method to calculate the final total bill for each customer. Here's an example usage:
