Here's the implementation of the Cashier class in Python:

```python
from typing import List

class Cashier:
    def __init__(self, n: int, discount: int, products: List[int], prices: List[int]):
        self.n = n
        self.discount = discount
        self.products = products
        self.prices = prices
        self.customer_count = 0

    def getBill(self, product: List[int], amount: List[int]) -> float:
        self.customer_count += 1
        bill = 0

        for i in range(len(product)):
            product_id = product[i]
            product_amount = amount[i]
            price = self.prices[self.products.index(product_id)]
            bill += price * product_amount

        if self.customer_count % self.n == 0:
            bill = bill * ((100 - self.discount) / 100)

        return bill
```

You can use the Cashier class as shown in the example:

```python
cashier = Cashier(3, 50, [1, 2, 3, 4, 5, 6, 7], [100, 200, 300, 400, 300, 200, 100])
print(cashier.getBill([1, 2], [1, 2]))  # Output: 500.0
print(cashier.getBill([3, 7], [10, 10]))  # Output: 4000.0
print(cashier.getBill([1, 2, 3, 4, 5, 6, 7], [1, 1, 1, 1, 1, 1, 1]))  # Output: 800.0
print(cashier.getBill([4], [10]))  # Output: 4000.0
print(cashier.getBill([7, 3], [10, 10]))  # Output: 4000.0
print(cashier.getBill([7, 5, 3, 1, 6, 4, 2], [10, 10, 10, 9, 9, 9, 7]))  # Output: 7350.0
print(cashier.getBill([2, 3, 5], [5, 3, 2]))  # Output: 2500.0
```

The `getBill` method takes the product and amount arrays as input and returns the final total of the bill with the discount applied (if applicable). The Cashier class keeps track of the customer count and applies the discount to every nth customer.
