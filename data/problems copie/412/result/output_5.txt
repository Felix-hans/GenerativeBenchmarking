Here's the implementation of the FizzBuzz problem in Python:

```python
class Solution:
    def fizzBuzz(self, n: int) -> List[str]:
        result = []
        for i in range(1, n + 1):
            if i % 3 == 0 and i % 5 == 0:
                result.append("FizzBuzz")
            elif i % 3 == 0:
                result.append("Fizz")
            elif i % 5 == 0:
                result.append("Buzz")
            else:
                result.append(str(i))
        return result
```

You can create an instance of the `Solution` class and call the `fizzBuzz` method to get the desired output. For example:

```python
solution = Solution()
print(solution.fizzBuzz(3))  # Output: ["1", "2", "Fizz"]
print(solution.fizzBuzz(5))  # Output: ["1", "2", "Fizz", "4", "Buzz"]
print(solution.fizzBuzz(15))  # Output: ["1", "2", "Fizz", "4", "Buzz", "Fizz", "7", "8", "Fizz", "Buzz", "11", "Fizz", "13", "14", "FizzBuzz"]
```

The `fizzBuzz` method takes an integer `n` as input and returns a list of strings following the FizzBuzz pattern.
