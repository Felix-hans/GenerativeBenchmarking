Here is the solution to the problem:

```go
type Allocator struct {
    memory []int
}

func Constructor(n int) Allocator {
    memory := make([]int, n)
    return Allocator{memory: memory}
}

func (a *Allocator) Allocate(size int, mID int) int {
    for i := 0; i <= len(a.memory)-size; i++ {
        if a.memory[i] == 0 {
            isBlockAvailable := true
            for j := i; j < i+size; j++ {
                if a.memory[j] != 0 {
                    isBlockAvailable = false
                    break
                }
            }
            if isBlockAvailable {
                for j := i; j < i+size; j++ {
                    a.memory[j] = mID
                }
                return i
            }
        }
    }
    return -1
}

func (a *Allocator) Free(mID int) int {
    count := 0
    for i := 0; i < len(a.memory); i++ {
        if a.memory[i] == mID {
            a.memory[i] = 0
            count++
        }
    }
    return count
}
```

Explanation:
- The `Allocator` struct maintains the memory array.
- The `Constructor` function initializes the memory array with all units being free.
- The `Allocate` function iterates over the memory array to find a leftmost block of `size` consecutive free memory units. If found, it assigns the given `mID` to each unit in the block and returns the first index of the block. If no such block exists, it returns -1.
- The `Free` function iterates over the memory array and frees all memory units with the given `mID`. It returns the number of memory units freed.

Note: This implementation assumes that `size` will always be less than or equal to the size of the memory array.
