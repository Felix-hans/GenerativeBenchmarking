To implement the above task, we can follow these steps:

1. Create an empty set to store distinct integers.
2. Iterate through each integer in the input array nums.
3. Convert each integer to a string and reverse its digits.
4. Convert the reversed string back to an integer.
5. Add the reversed integer to the set of distinct integers.
6. Finally, return the length of the set of distinct integers.

Here is the implementation of the Solution class:

```python
from typing import List


class Solution:
    def countDistinctIntegers(self, nums: List[int]) -> int:
        distinct_integers = set()

        for num in nums:
            reversed_num = int(str(num)[::-1])
            distinct_integers.add(reversed_num)

        return len(distinct_integers)
```

You can then create an instance of the Solution class and call the `countDistinctIntegers` method, passing in the input array as an argument.
